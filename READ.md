# 💰 Agente Financeiro com IA

Dashboard interativo para análise de gastos com integração de IA.

## 🚀 Funcionalidades

- Upload de CSV
- Visualização de dados
- Gráficos automáticos
- Métricas financeiras
- Perguntas com IA

## 🛠 Tecnologias

- Python
- Streamlit
- Pandas
- Matplotlib
- OpenAI

## ▶️ Como rodar

```bash
pip install -r requirements.txt
streamlit run src/app.py
